# 🚀 Демо инструкции для клиента

## Как запустить проект локально:

### 1. Установка зависимостей:
```bash
npm install --legacy-peer-deps
```

### 2. Запуск проекта:
```bash
npm run dev
```

### 3. Откройте браузер:
```
http://localhost:5173
```

## 🎯 Что клиент увидит:
- ✅ Полный интерфейс Promtly AI
- ✅ Чат с AI ботом
- ✅ Все функции разработки
- ✅ Генерация кода
- ✅ Файловый менеджер
- ✅ Терминал

## 🔑 Для работы AI нужно добавить API ключи:
1. Создайте файл `.env.local`
2. Добавьте ваши ключи:
```
ANTHROPIC_API_KEY=your_key_here
OPENAI_API_KEY=your_key_here
```

## 📱 Альтернатива - используйте ngrok для публичного доступа:
```bash
npx ngrok http 5173
```
Получите публичную ссылку для демонстрации клиенту удаленно.
